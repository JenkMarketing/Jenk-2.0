# Business Model — Keystone/Jenk Agency

## What We Sell

A pre-built GHL marketing system. NOT custom work. NOT an ads agency.

Price: $197-297/month recurring forever.
Entry point: $97/month website only → upsell to full system on launch call.

## The 5 Core Features

1. SEO-optimised website (10-20 pages, chat widget → text, quote form → text, review widget)
2. Missed call text back (60% of calls go unanswered — auto-texts quote link instantly)
3. 5-star Google review system (4-5 stars go public, 1-3 stars go private, 4 reminders over 30 days)
4. Remarketing sequence (hits past customers every 6 weeks with discount + referral ask)
5. Business phone number (toll-free, all calls/texts → GHL mobile app)

## The Stickiness Mechanism

Everything routes back to the GHL mobile app. Client uses app daily = they never churn.

Mental model: stack of client blocks. Red = churned. Green = sticky. Replace reds, keep adding greens.

## Ideal Client

High volume trades ONLY: landscapers, dog groomers, carpet cleaners, handymen, plumbers (service calls), electricians, HVAC, pest control, painters, pool cleaners, pressure washers.

AVOID: kitchen remodelers, roofers, bathroom remodelers (1-4 jobs/month, automations don't fire enough).

Sweet spot: established business, knows they need website + reviews, just doesn't know how to set it up.

## Sales Process

1. Lead comes in (cold call, SMS, ad, referral)
2. Book 10-min phone consult — cap to 48-72hr window
3. Send Trojan Loom VSL immediately after booking (5-6 min screen recording of demo account)
4. Do NOT take call if they haven't watched the VSL
5. Sales call 20-35 mins — they show up pre-sold
6. Get card on file on the call — never let them off without it
7. Free trial (5-7 days) if hesitant — card still required

## Closing Rules

- Never let anyone off the phone without a credit card
- Fill onboarding form together while on the call
- Offer free trial to get card — "won't be charged for 7 days"
- "Let me think about it" = they got confused = simplify your pitch

## Delivery (A-Z)

1. Create sub-account from SSA snapshot
2. Buy toll-free number → configure forwarding
3. Fill all custom values (service areas, colors, phone numbers, GMB link, review link)
4. Connect Google + GMB
5. Set up reputation management (auto-respond 4+ stars, 5-min wait)
6. Build website (home, service pages, service area pages, FAQ, about, footer)
7. Buy GoDaddy domain → connect to GHL → generate sitemap
8. Test everything on desktop + mobile
9. Launch call: walk client through mobile app, demo all features
10. Goal: client using mobile app daily = they never leave

## Pricing Tiers

- $97/month — website only (foot in door)
- $197/month — core system
- $297/month — full system + extras

Never sell time and talent. Sell automation.
